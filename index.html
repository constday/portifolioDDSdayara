Janela MDI

O projeto tem com baso na criação de um o formulário principal aonde aparecerá como um contêiner MDI. Com MenuStrip, você pode criar janelas separadas para o sistema de cadastro de uma escola, tudo dentro do formulário principal. Operação A janela principal funciona como um contêiner, mantendo todas as outras janelas juntas. Cuida de organizar e orientar as ações das janelas filhas. Janelas filhas: são como pequenas janelas que aparecem dentro do contêiner principal, oferecendo mais opções e informações. Eles podem ser abertos, fechados, movidos e redimensionados no formulário principal. Essa estrutura é ótima para quando você tem muitas coisas diferentes para fazer ao mesmo tempo, como trabalhar em documentos diferentes ou usar partes diferentes de um aplicativo. Ajuda a manter tudo organizado e torna mais fácil alternar entre coisas diferentes sem se confundir..
